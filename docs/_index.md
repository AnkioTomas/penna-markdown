## 站点约定

| 文件          | 作用                                        |
| ------------- | ------------------------------------------- |
| `_sidebar.md` | 当前目录侧栏：分组标题 + `- [显示名](路径)` |
| `_index.md`   | 目录入口页（优先于 `index.md`）             |
